# JoinMe Admin Panel

Админ-панель для управления приложением JoinMe.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Админ-панель будет доступна по адресу: http://localhost:3001

## Настройка

### Переменные окружения

Создайте файл `.env` в корне проекта:

```env
VITE_API_URL=http://localhost:3000/api
```

### Настройка бэкенда

В файле `.env` бэкенда (`joinme-backend/.env`) настройте учетные данные админа:

```env
ADMIN_USERNAME=admin
ADMIN_PASSWORD=admin123
ADMIN_TOKEN=admin-secret-token-change-in-production
```

**ВАЖНО**: В продакшене обязательно измените эти значения на безопасные!

## Вход

По умолчанию:
- Логин: `admin`
- Пароль: `admin123`

## Функционал

- **Все события**: Просмотр, детали и удаление событий
- **Пользователи**: Просмотр, детали и удаление пользователей
- **Чаты**: Просмотр всех чатов, сообщений и удаление чатов
- **Статистика**: Общая статистика приложения, события по городам, новые пользователи

## Безопасность

- Все запросы к API защищены токеном авторизации
- Токен хранится в localStorage
- При неавторизованном доступе происходит редирект на страницу входа
